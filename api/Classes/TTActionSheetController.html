Conforms to UIActionSheetDelegate

Overview

A view controller that displays an action sheet.

This class exists in order to allow action sheets to be displayed by TTNavigator, and gain all the benefits of persistence and URL dispatch.

By default this controller is not persisted in the navigation history.
